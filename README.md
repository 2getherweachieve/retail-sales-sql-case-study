# Retail Sales SQL Case Study

## Project Overview
This project analyses a retail sales transactions dataset using SQL to uncover business insights related to sales performance, customer behaviour, profitability, and time-based trends.

The case study simulates a real business environment where management wants answers to questions such as:
- How are sales and profits performing?
- Which categories and customers drive revenue?
- What time periods generate the highest sales?
- How can customers be segmented for business decisions?

---

## Dataset Description
Table: `retails_sales`

Columns:
- transactions_id
- sale_date
- sale_time
- customer_id
- gender
- age
- category
- quantity
- price_per_unit
- cogs
- total_sale
